# Projet

Application web pour telle fonctionnalité. 

## Getting Started

Ces instructions permettent d'executer une copie du projet en local sur votre poste de travail pour le développement et les tests. Référez-vous à la section "Déploiement" pour les étapes à suivre pour déployer le projet en production.

### Pré-requis

Pour executer en local le projet Projet, vous devez au préalable installer :

```
Git
```

### Installation

Voici les étapes à suivre pour avoir un environnement de développement et de test opérationnel :


```
Installer Git sur votre poste de travail
```



## Executer les tests

```
Modifier le fichier readme.md pour a ajouter votre nom d’utilisateur github dans la liste des auteurs.
```


## Deployment

Voici les étapes à suivre pour déployer en production :

```
Faire un pull request du fichier readme.md modifié
```
